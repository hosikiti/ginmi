Ginmi Quick Start

Jump to the right window fast.

Ginmi is a lightweight Mac window switcher. Open it, type a few letters, and go straight to the exact window you want instead of cycling through everything.

Default quick switch Tap Command + Tab quickly to jump back to the previous window. Hold it for about 200 ms to open the switcher instead.

What Ginmi Searches

  • Running app names
  • Window titles
  • Your recent usage, so common picks rise to the top
  • Installed apps, if there is no running window match

Install

1
Open the DMG Double-click the `Ginmi.dmg` file you received.
2
Move the app to Applications Drag `Ginmi.app` into your `/Applications` folder.
3
Open it from Applications If macOS warns that the app is from an unidentified developer, right-click `Ginmi.app` and choose Open once.
4
Allow Accessibility access Ginmi needs this so it can see and focus windows across apps and spaces.
If you already enabled Accessibility but Ginmi still asks again, remove `Ginmi` from the Accessibility list, add `/Applications/Ginmi.app` again, then relaunch it.

Use

  • Use the menu bar icon to open the search panel.
  • Type 2 to 4 letters from the app name or window title.
  • Use ↑ and ↓ to move through results.
  • Press Return to jump to the selected window.
  • Press Esc to cancel.
  • Use Shift + Q in the switcher to quit the selected app.

Settings

  • Open Ginmi’s settings from the menu bar icon.
  • Adjust the Cmd+Tab quick-switch delay if `200 ms` feels too fast or too slow.
  • Reset learned shortcuts if results start feeling strange.
